Descrição do Projeto: Controle de Motor de Passo Utilizando Machine Learning e Visão Computacional

Introdução

O projeto tem como objetivo desenvolver um sistema que utilize técnicas de Machine Learning e visão computacional para controlar um motor de passo com base no movimento do rosto de uma pessoa específica. Esse sistema pode ser aplicado para melhorar a acessibilidade de pessoas com deficiência (PCD), como o controle de cadeiras de rodas, a ligação/desligação de aparelhos elétricos (motores, bombas, carros elétricos, etc.) e outras automações.

Tecnologias Utilizadas

  • Linguagem de Programação: Python
  • Bibliotecas de Visão Computacional: OpenCV, HaarCascade
  • Frameworks de Machine Learning: PyTorch
  • Hardware: Sistema embarcado (e.g., Raspberry Pi ou Arduino), motor de passo

Funcionamento do Sistema

  1. Aquisição de Dados pela Câmera:
    • Uma câmera captura o vídeo em tempo real.
    • Os frames capturados são processados utilizando o OpenCV.
  2. Reconhecimento Facial:
    • Utiliza-se o algoritmo HaarCascade para detectar o rosto no frame.
    • Um modelo de deep learning treinado com PyTorch identifica se o rosto pertence ao indivíduo específico autorizado a controlar o sistema.
  3. Análise de Movimento do Rosto:
    • Algoritmos de rastreamento facial (como dlib ou mediapipe) analisam o movimento do rosto.
    • O movimento detectado (para cima, baixo, esquerda, direita) é convertido em comandos.
  4. Controle do Motor de Passo:
    • Um microcontrolador recebe os comandos de movimento.
    • O motor de passo é controlado para executar os movimentos desejados com base na direção do rosto.
  5. Segurança e Automação:
    • O sistema é projetado para responder apenas ao rosto do usuário cadastrado, aumentando a segurança.
    • Movimentos mais complexos, como o rastreamento da íris, podem ser implementados como um upgrade, permitindo comandos mais precisos.

Aplicabilidades

  • Acessibilidade: Controle de cadeiras de rodas ou outros dispositivos assistivos.
  • Automação Residencial: Ligação/desligação de aparelhos elétricos.
  • Indústria: Controle de máquinas ou equipamentos em ambientes industriais.

Benefícios

  • Inclusão e acessibilidade para PCDs.
  • Automação inteligente e personalizada.
  • Redução da necessidade de controles físicos convencionais.

Possíveis Extensões

  • Integração com comandos de voz para funcionalidades adicionais.
  • Implementação de redes neurais mais complexas para reconhecimento de gestos faciais detalhados.
  • Conexão com sistemas IoT para controle remoto.
